# Abraham Public College

Single-page website for Abraham Public College, Lucknow. Built with React, Vite and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

Then open the printed local URL (usually `http://localhost:5173`).

## Build for production

```bash
npm run build
npm run preview
```

## Editing content

All page text (headings, copy, form labels, contact details, etc.) lives in one place: `src/content.js`. Update that file to change wording without touching any component code.

Section components are in `src/components/`. Placeholder image blocks (`ImagePlaceholder`) mark where real photos should go.
